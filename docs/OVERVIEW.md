# Lingvago — Обзор проекта

## Назначение

Lingvago — PWA-приложение для подготовки к экзамену A2 по европейскому португальскому языку. Приложение использует алгоритм интервального повторения FSRS для эффективного запоминания лексики и предоставляет различные режимы обучения.

## Целевая аудитория

Один пользователь (автор). Приложение не имеет серверной части, аутентификации или многопользовательских функций. Все данные хранятся локально в браузере.

## Ключевые принципы

| Принцип | Описание |
|---------|----------|
| **Offline-first** | Приложение полностью работает без интернета. Service Worker кэширует все ресурсы, данные хранятся в IndexedDB |
| **Mobile-first** | Основной сценарий использования — телефон. Дизайн адаптируется от мобильного к десктопу |
| **No backend** | Нет сервера, API, базы данных на сервере. Вся логика и данные — на клиенте |
| **Расширяемость** | Новые режимы обучения добавляются с минимальными изменениями существующего кода |
| **Минимализм** | Никакой геймификации, UI-библиотек, лишних зависимостей |
| **Локализация** | UI на русском и английском (i18next). Учебный контент — отдельные программы по языковым парам (PT→RU для v1). Архитектура готова к добавлению новых языков |

## Приоритеты реализации

### v1 — must have
- VocabularyMode с FSRS (multiple choice + flip card)
- Колоды (Decks) с isActive
- CRUD слов в колодах
- Study-экран + session summary
- Dashboard
- Офлайн PWA (Service Worker + manifest)

### v1 — nice to have
- NumbersMode
- CSV-импорт слов
- Stats экран
- Темы (dark/light)
- i18n UI (можно начать только с RU, EN добавить позднее)

### Future (после v1)
- Переключение studyLanguage (PT→EN)
- JSON export/import данных
- Полная i18n (английский UI)

## Что НЕ входит в scope

- Серверная часть / API / синхронизация между устройствами
- Аутентификация / авторизация
- Геймификация (стрики, ачивки, XP)
- Text-to-speech / распознавание речи (может быть добавлено позднее)
- Грамматические упражнения (может быть добавлено как отдельный LearningMode)
- UI-библиотеки (Material UI, Chakra, и т.д.)
